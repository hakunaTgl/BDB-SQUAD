# BDB-SQUAD Repository

This repository contains the **AetherOS** project - an open-source, multimodal creative AI system.

## 📂 Project Location

All project files are located in the **`gengen/`** directory.

## 🚀 Quick Start

```bash
cd gengen/
./studio.sh
```

Or run the minimal demo:

```bash
cd gengen/
python3 aetheros_minimal.py
```

## 🎯 What is AetherOS?

AetherOS is a fully open-source, multimodal creative AI system that democratizes imagination—**no coding required**.

**Key Features:**
- 🧠 Neural Graph Memory
- 🎭 Affective Reasoning
- 🤖 Agent Orchestration
- 🎨 Multimodal Generation (Video, Audio, Text)
- 🎤 Audio-Driven Animation
- 📽️ Storyboarded Video

## ⚡ System Requirements

- Python 3.8+
- 4 GB RAM minimum
- 1 GB disk space (minimal mode)
