## Vision

AetherOS is a fully open-source, multimodal creative AI system with integrated Hermes LLM for intelligent text generation. Democratize imagination—no coding required.

---

## New: Hermes LLM Integration

AetherOS now features the **unfiltered Hermes model** as its core language model, providing:

- 🧠 **Intelligent Prompt Enhancement** - Expands creative concepts with rich detail
- 📝 **Script Generation** - Creates properly formatted, compelling scripts  
- 🎬 **Story Decomposition** - Breaks narratives into structured scenes
- 🤖 **Meta-Prompting** - Generates intelligent task execution strategies
- 🔍 **Content Analysis** - Evaluates quality, ethics, and themes

See `LLM_SETUP.md` for setup instructions.

---

## How to Use

- See `LAUNCH_GUIDE.md` for all launch options and quick start.
- See `LLM_SETUP.md` for Hermes model setup (optional but recommended).
- Use the web interface for full control: `python3 aetheros_interface.py`
- Use the minimal demo for instant results: `python3 aetheros_minimal.py`
- Use the launcher for guided setup: `./studio.sh`
