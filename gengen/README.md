**Minimal Mode (Terminal Interface):**
- Python 3.8+ only
- Zero dependencies
- Works anywhere!

**Web Interface:**
- Python 3.8+
- Gradio (~50 MB)
- Runs in browser

**Full System (Future):**
- Python 3.11+

- 8GB+ VRAM (24GB recommended for full features)

```bash

# Initialize Neural Graph Memory

# Run the orchestrator
python src/orchestrator/main.py
```
