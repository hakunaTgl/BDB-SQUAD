## 🎯 Vision

AetherOS is a fully open-source, multimodal creative AI system that democratizes imagination—**no coding required**.

Create professional-quality stories, videos, and audio through natural language prompts. Built on ethical AI principles with complete transparency and user control.

---

## 🚀 Quick Start

**Three ways to get started:**

1. **One-Click Launch** (Recommended)
   ```bash
   ./studio.sh
   ```

2. **Web Interface** (Best Experience)
   ```bash
   pip install gradio
   python3 aetheros_interface.py
   ```

3. **Terminal** (Zero Dependencies)
   ```bash
   python3 aetheros_minimal.py
   ```

**First time?** → Read `START_HERE.md` for a quick introduction!

## 🏗️ Project Structure

```
gengen/
├── src/                         # Core source code
│   ├── core/                   # Core systems (NGM, APU)
│   ├── agents/                 # Agent implementations
│   ├── orchestrator/           # RMP orchestrator
│   └── pipelines/              # Media pipelines
├── aetheros_interface.py       # Web interface (Gradio)
├── aetheros_minimal.py         # Terminal interface (no deps)
├── run_aetheros.py             # Quick demo script
├── launch_studio.py            # Smart launcher
├── requirements.txt            # Dependencies
└── *.md                        # Documentation
```

## ⚙️ System Requirements

**Minimum:**
- Python 3.8+
- 4 GB RAM
- 1 GB disk space (minimal mode)

**Recommended:**
- Python 3.10+
- 8-16 GB RAM
- 10 GB disk space
- GPU optional (CPU mode available)
